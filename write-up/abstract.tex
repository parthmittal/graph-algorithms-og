\begin{abstract}

We present a serial implementation to compute Betweenness Centrality of an
undirected, unweighted, biconnected graph, using a technique presented in
\cite{pachorkar16}, and measure the improvements it offers over Brandes
Algorithm (see \cite{brandes01}).

\vspace{2in}
Keywords: high-performance-computing, pain
\end{abstract}
